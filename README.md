# DormLock

A multilingual campus-lore joke site built with React and Vite. This is not a locksmith service.

## Local development

```bash
npm install
npm run dev
```

## Vercel deployment

1. Verify locally with `npm run build`.
2. Push the project to a GitHub repository.
3. Import that repository in Vercel and keep the detected Vite defaults.
4. Vercel provides a free `project-name.vercel.app` domain.

Supabase environment variable placeholders are documented in `.env.example`; the current review form runs entirely in local React state.
